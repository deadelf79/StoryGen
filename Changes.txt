Генератор сюжета, список изменений.
Версия 1.0.3.
Изменения:
 - Заменен компонент для вывода текста на новый, поддерживающий форматирование
 - Кнопка "удалить вариант" по умолчанию выключена и включается только, если выбран один из вариантов
 - Добавлена возможность множественного выбора вариантов для удаления из списка вариантов событий
 - Кнопка "удалить вариант" позволяет удалять сразу несколько вариантов из списка
 - Изменены выравнивания скроллбара для количества событий и подписей к нему, теперь они не висят "в воздухе"
Добавлено:
 - Начата работа над генератором персонажей
   - Добавлены вкладки для генераторов - "события" и "персонажи" (кнопка "генератор персонажей" теперь больше не нужна)
 - Добавлена панель для редактирования текста, чтобы ее активировать, нужно поставить соответствующую галочку
 - Начата работа над английской локализацией
 - Форматирование текста
	- Полужирный
	- Курсив
	- Подчеркивание
	- Зачеркивание
 - Добавлена панель инструментов для персонажей
 - Возможность вручную изменять размеры окна
Версия 1.0.2.
Изменения:
 - Кнопка "добавить вариант" включает кнопку "запустить генератор", если вариантов в списке от трех и более
 - Кнопка "удалить вариант" выключает кнопку "запустить генератор", если вариантов в списке менее трех
 - Списки вариантов обновлены:
	- Роман ужасов (14 новых событий)
	- Любовный роман (30 новых событий)
	- Космическая фантастика (13 новых событий)
	- Киберпанк (44 новых события)
 - Текстовому полю для ввода имени нового варианта события добавлена подсказка для того случая, когда оно пустое
 - Добавлено открытие страницы блога разработчика в окне браузера по умолчанию при нажатии ссылки (она находится слева внизу)
Добавлено:
 - На форму "Выбор шаблона" добавлена панель выбора файла с вариантами событий
 - Туда же: список ранее использованных файлов
 - Дополнительная форма для показа ошибки, если не выбран ни один из файлов из файловой системы при выборе вариант загрузки "Пользовательские" для формы "Выбор шаблона"
 - В ту же форму добавлена небольшая пасхалка
 - На главную форму добавлена кнопка "Информация о программе"
 - В форму "Общая информация" добавлена информация о программе, версии и количестве встроенных жанров/событий.
